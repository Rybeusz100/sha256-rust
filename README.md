# SHA-256 calculator
## Usage:
```
sha256-rust.exe [OPTIONS] --files <files>...
```
## Options:
```
-f, --files <files>... - Filenames separated by spaces
```
```
-h, --help - Print help information
```
```
-m, --memory <memory> - How much memory should the program use as a buffer (in kilobytes)
```
